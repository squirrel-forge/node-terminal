# node-terminal

A node commandline abstraction, application wrapper and commands.

## Installation

```
npm i @squirrel-forge/node-terminal

```

## Usage

```
const nt = require( '@squirrel-forge/node-terminal' );
// nt.Class
```

 - Application
 - Command
 - HelpCommand
